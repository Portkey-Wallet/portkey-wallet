# Getting Start
## Code style

pre commit: husky + eslint + commit lint + prettier + Typescript

after publish: SuperLinter + eslint

## Installing
```
yarn
```

## Run
```
yarn extension-did: dev
```

## build
```
yarn extension:did pro
```
